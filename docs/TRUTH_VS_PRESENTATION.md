# Axiom — Truth vs Presentation

Defines the hard boundary between simulation truth and presentation.

---

## Core Rule

C++ owns truth.
C# owns interpretation.

---

## Truth (C++)

- World state
- Simulation rules
- Determinism
- Commands
- Save/load

---

## Presentation (C#)

- Rendering
- UI
- Unit conversion
- Tone and localization

---

## Forbidden

- C++ calling into C#
- UI logic in simulation
- Display units in truth
